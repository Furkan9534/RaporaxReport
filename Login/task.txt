Authentication sistemi, Microsoft teknolojileri kullanılarak ASP.NET Core Web API üzerinde geliştirilecektir; sistem, kullanıcı ve rol verilerini merkezi bir kimlik veritabanında (tek DB) tutacak ve çoklu firma (multi-tenant) yapısını destekleyecek şekilde tasarlanacak; üretilen JWT içerisinde ilgili kullanıcının tenant/company bilgisi taşınacak, sistem stateless çalışacak ve kullanıcı oturum bilgilerini server memory üzerinde tutmayacaktır; kullanıcı giriş işlemleri, şifre değiştirme, e-mail ile şifre sıfırlama ve rol bazlı yetkilendirme (Roller : Rapor görüntüleyici, Rapor oluşturucu, Admin) sağlayacak, JWT tabanlı authentication kullanılacak, parola politikaları tanımlanacak, login endpoint’lerine rate limiting uygulanacak ve token üretimi güvenli şekilde yapılacaktır; sistem güvenlik açısından OWASP Top 10 riskleri dikkate alınarak geliştirilecek, tüm veri erişimi ORM üzerinden gerçekleştirilecek, input validation uygulanacak ve hata mesajları güvenli olacak şekilde tasarlanacaktır; core authentication işlevleri için unit ve integration testler yazılacak, JWT claim yapısı ve tüm endpoint’ler dokümante edilecek, diğer sistemlerle entegrasyon yalnızca HTTP ve JWT doğrulama mekanizması üzerinden sağlanacak ve container uyumlu olacak şekilde Docker ile paketlenebilir biçimde hazırlanacaktır.
